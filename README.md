# Unkown Song Recognition System

A university final year project titled Unknown Song Recognition on Mobile Phone by Audio Fingerprint

## Features
* **Audio Recognition:** Accurately identifies song through direct audio input.
* **Humming Recognition:** Supports user humming as well. 
* **Spotify Integration:** Fetches track metadata, album art, and song recommendations using Spotify API 
* **Mobile Application**

## Tech Stack

**Client Side (Android)**
* Kotlin
* Android Studio
* Retrofit, OkHttp

**Server-side and Database**
* Node.js, Express.js
* PostgreSQL

**Third Party Integration**
* ACRCloud SDK
* Spotify Web API
