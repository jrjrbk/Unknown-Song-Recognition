# Unkown Song Recognition System

A university final year project titled Unknown Song Recognition on Mobile Phone by Audio Fingerprint

## Features
* **Audio Recognition:** Accurately identifies song through direct audio input.
* **Humming Recognition:** Supports user humming as well. 
* **Spotify Integration:** Fetches track metadata, album art, and song recommendations using Spotify API 
* **Gamification:** Users can gain points, earn badges, and track their progress based on the number of songs they've identified
* **Playlist:** Automatically adds or remove identified songs to playlists, with features to generate recommendations based on playlist content.

## Tech Stack

**Client Side (Android)**
* Kotlin
* Android Studio
* Retrofit, OkHttp
* Picasso (Image Caching)

**Server-side and Database**
* Node.js, Express.js
* PostgreSQL
* Google Cloud Storage

**Third Party Integration**
* ACRCloud SDK
* Spotify Web API
